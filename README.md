# 🧠 Lógica de Programação & Algoritmos com Java

Repositório dedicado ao estudo de estruturas de dados, algoritmos clássicos e resolução de desafios de lógica. Este projeto serve como portfólio de evolução técnica durante minha graduação em **Sistemas de Informação**.

---

## 🚀 Tecnologias e Ferramentas
* **Linguagem:** Java 17+
* **IDE:** VS Code / IntelliJ IDEA
* **Paradigma:** Orientação a Objetos (POO)

---

## 📂 Estrutura do Repositório

Como este projeto não utiliza gerenciadores de dependência (Maven/Gradle), a estrutura segue o padrão de pacotes nativo do Java:

```text
/src
 ├── algorithms/      # Algoritmos de ordenação, busca, etc.
 ├── arrays/          # Manipulação de vetores e matrizes
 ├── strings/         # Desafios envolvendo textos
 └── structures/      # Implementações de Pilhas, Filas e Listas

## 📊 Tabela de Desafios

Abaixo, os exercícios resolvidos e sua análise de eficiência:

| Exercício | Categoria | Complexidade | Status |
| :--- | :--- | :--- | :---: |
| Inversão de String | Strings | O(n) | 🚧 |
| Busca Binária | Algoritmos | O(log n) | ⏳ |
| Soma de Dois (Two Sum) | Arrays | O(n) | ⏳ |

> **Nota:** Para cada desafio, busco otimizar o código visando a melhor performance em tempo e espaço utilizando a notação **Big O**.
